Enhanced Ecommerce With a Custom Coded CMS

MCF Assisted Conversions blended in GDS

At the time I first met that digital marketing agency, they already had a Google Data Studio dashboard to evaluate lead generating efforts broken down by marketing channel for a client of theirs. A simple table with common GA data, no big deal. The problem was they tried a few times to no success to extend the table with a single additional metric and were asking if I could help.

What made the project challenging was that the requested metric was Assisted Conversions from GA Multi-Channel Funnel report. To make things a bit more fun, they had a custom channels grouping, but only inside GDS.

To deliver the solution while having in mind all project limitations was quite an exciting task. Here's what I did:

  • set up automatic pulls of Assisted Conversions into Google Sheets using a separate MCF API;
  • imported that sheet as a data source into GDS while applying unified custom channel grouping to both itself and GA source;
  • joined GA and MCF data by creating new blended data source;
  • extended the table with MCF Assisted Conversions metric which was linked with other marketing channels data from GA;
  • and finally, fixed table's summary row to overcome GDS issue when calculating totals from blended data sources.

The project was a success: added metric has led to a better, more holistic understanding of marketing channels performance.

Simple Metric Calculation for GDS Dashboard

How complex could be dividing two numbers and showing the difference? You might be thinking, even toddler should handle that. Well, when it comes to GDS, you may have to take a scenic route.

A client of mine, he manages CRO business, needed a simple way of keeping his clients informed on the progress of optimization efforts. Clearly, a new way should be better that emailing monthly reports, so I came up with a pretty basic dashboard, showing how A\B tests are going in real-time. One of the metrics we agreed on was % delta (difference) between A and B variants' performance. But due to some GDS limitations I had to create the following workaround.

  1. First, I've set up scheduled pulls of data from Google Analytics into the Google Sheets, reference copied that data on a separate sheet and added delta calculations for a number of metrics.
  2. With calculations handled I was able to import the sheet as a new data source in GDS and prettify it with formatting. So far so good.
  3. Finally I could display delta on the dashboard. Hooray.
